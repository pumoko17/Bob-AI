# Contoh Pertanyaan User - Bob AI

Berikut adalah 10 contoh pertanyaan yang dapat diajukan user kepada Bob AI:

## Pertanyaan Terkait Dokumen Proyek

1. **Apa itu Bob AI dan apa fungsinya?**
   - Target: Mengetahui definisi dan fungsi Bob AI dari dokumen S01

2. **Apa saja komponen utama dalam workflow RAG yang digunakan Bob AI?**
   - Target: Memahami komponen RAG dari dokumen S06

3. **Bagaimana cara kerja query expansion dalam sistem Bob AI?**
   - Target: Memahami mekanisme query expansion dari dokumen S06

4. **Apa perbedaan antara semantic search dan BM25 reranking?**
   - Target: Memahami perbedaan teknik retrieval dari dokumen S06

5. **Bagaimana Bob AI mencegah halusinasi dalam memberikan jawaban?**
   - Target: Memahami guardrails dari dokumen S02

6. **Apa saja langkah-langkah dalam simulasi alur kerja Bob AI?**
   - Target: Memahami workflow dari dokumen S03

7. **Siapa target user dari Bob AI dan apa kebutuhan mereka?**
   - Target: Memahami target user dari dokumen S01

## Pertanyaan Terkait Dataset University Rankings

8. **Universitas mana yang mendapat peringkat terting dalam CWUR ranking tahun terbaru?**
   - Target: Query dataset university rankings

9. **Apa perbedaan metrik penilaian antara Times Higher Education dan Shanghai ranking?**
   - Target: Membandingkan metrik ranking

10. **Bagaimana skor alumni employment mempengaruhi peringkat universitas dalam CWUR?**
    - Target: Memahami faktor penilaian

## Catatan untuk Demo
- Gunakan minimal 3-5 pertanyaan saat demo
- Pastikan pertanyaan mencakup berbagai aspek: definisi, teknis, dan evaluasi
- Siapkan dokumen yang relevan sebelum mengajukan pertanyaan
